Installation

Build it Yourself

Clone the Repository
git clone https://github.com/TerrificTable/Yasashii-Rewrite
Open the folder
cd Yasashii-Rewrite
Setup Gradle
gradlew
Build the Jar
gradlew build
You should be able to see a new folder called "build"
In which a folder called "libs" should be within a file called "Yasashii-v.1.0.3.jar" now just take the file and put it inside your mods folder and run Minecraft forge
