ViewBot.

Introduction

This is a ViewBot for YouTube, TikTok or whatever you want, it was originaly made to not be open src but i thought about it again and made it open src

ViewBot on Github

The ViewBot has its own UI and it doesnt need much librarys to install.

Setting it up

just git clone the repository.

The Github Repository

Use the Command:

git clone https://github.com/TerrificTable/YouTube-and-TikTok--View-Bot

In your CMD, after that execute:

cd YouTube-and-TikTok--View-Bot

after that you execute this command:

python -m pip -r requirements.txt

and wait, then you should be able to execute:

python viewbot.py

without any Errors

Functions

This ViewBot is made by: TerrificTable and Dom

And it has these Functions:

- ViewBot v.1 (uses tabs to BotViews)

- ViewBot v.2 (uses ChromeDriver to BotViews)

- ChannelBot (goes on a given channel and clicks on every Video it can find to BotViews)

It also has a random delay (between 0 and 60sec) added to a given delay
